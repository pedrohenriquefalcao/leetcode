# LeetCode Solutions

Repositório contendo soluções para problemas do [LeetCode](https://leetcode.com/) implementadas em **Java**.

## 📋 Sobre

Este projeto é uma coleção de soluções otimizadas para problemas de algoritmos e estruturas de dados do LeetCode. Cada solução inclui:

- ✅ Código-fonte implementado
- 📖 Descrição do problema
- 💡 Abordagem e explicação da solução

## 🗂️ Estrutura do Projeto

```
src/
├── Main.java                                          # Arquivo principal
├── palindrome_number/
│   ├── Main.java                                      # Solução do problema
│   └── PROBLEM.md                                     # Descrição do problema
└── Number_of_Strings_That_Appear_as_Substrings_in_Word/
    ├── Main.java                                      # Solução do problema
    └── PROBLEM.md                                     # Descrição do problema
```

## 🎯 Problemas Resolvidos

### 1. **Palindrome Number**
Verificar se um número inteiro é um palíndromo.
- **Dificuldade:** Fácil
- **Arquivo:** `src/palindrome_number/`

### 2. **Number of Strings That Appear as Substrings in Word**
Contar quantas strings de um array aparecem como substrings em uma palavra.
- **Dificuldade:** Fácil
- **Arquivo:** `src/Number_of_Strings_That_Appear_as_Substrings_in_Word/`

## 🚀 Como Usar

### Pré-requisitos
- Java 8 ou superior
- IntelliJ IDEA (recomendado) ou qualquer IDE Java

### Executando uma Solução

1. Abra o projeto em sua IDE favorita
2. Navegue até o arquivo `Main.java` da solução desejada
3. Execute o programa

## 📚 Informações Adicionais

Cada problema possui um arquivo `PROBLEM.md` na sua respectiva pasta com:
- Descrição completa do problema
- Exemplos de entrada e saída
- Restrições
- Explicação da abordagem
